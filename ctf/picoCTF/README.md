# picoCTF

This directory contains my solutions and notes from picoCTF challenges.

picoCTF is a free computer security education program created by security experts at Carnegie Mellon University.

## Challenge Categories

### General Skills
Basic skills and general challenges covering various aspects of cybersecurity fundamentals.

### Forensic
Digital forensics challenges involving file analysis, image forensics, memory analysis, and more.

### Cryptography
Challenges related to encryption, decryption, ciphers, and cryptographic protocols.
